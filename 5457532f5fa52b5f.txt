Инструкция по запуску:
1. Открываем командную строку
2. Копируем путь до файлов с ботом
3. Используем команды:
cd Скопированынй путь
node main.js
ПРИМЕР:
cd C:\Users\Desktop\bot_files
node main.js


Пример токенов:
В целом, все токены должны выглядеть так: 
tokens : [
"TOKEN1",
"TOKEN2",
"TOKEN3"
]
Примечание: После "TOKEN3" запятая не ставится! Это важно!
